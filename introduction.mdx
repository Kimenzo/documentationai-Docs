## Platform Overview

Lorenzo Otieno Documentation provides a centralized space to organize, manage, and publish your project documentation. You create structured pages, embed code examples, and collaborate seamlessly with your team. This default documentation hub supports MDX for rich, interactive content including components like cards, steps, and tabs.

## Quick Start

Set up your documentation space in minutes. Follow these steps to create your first page.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at the dashboard and verify your email.
  </Step>
  <Step title="Initialize Project" icon="folder-plus">
    Run the setup command in your terminal.
    
    <CodeGroup tabs="npm,yarn">
      ```bash
      npm create lorenzo-docs@latest my-project
      ```
      ```bash
      yarn create lorenzo-docs my-project
      ```
    </CodeGroup>
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Edit `introduction.mdx` with your YAML frontmatter and content.
  </Step>
  <Step title="Preview & Deploy" icon="rocket">
    Start the dev server: `npm run dev`, then deploy with `npm run deploy`.
  </Step>
</Steps>

<Expandable title="Advanced Configuration" default-open="false">
  Customize your site by editing `lorenzo.config.js`:
  
  ```javascript
  export default {
    brandColor: '#3B82F6',
    nav: [
      { title: 'Home', href: '/' },
      { title: 'API', href: '/api' }
    ]
  };
  ```
  
  Integrate with GitHub for automatic deployments on push.
</Expandable>
